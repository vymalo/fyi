# Vymalo FYI

This is the service behind the link vym.fyi, a url shortener.

## Docs

- Developer guide: `docs/README.md`
- Architecture: `docs/arc42.md`

## Getting started

- Install dependencies: `pnpm install`
- Run repo tasks via Turborepo (once apps/packages are scaffolded):
  - Lint: `pnpm lint`
  - Type-check: `pnpm typecheck`
  - Test: `pnpm test`
  - Build: `pnpm build`
